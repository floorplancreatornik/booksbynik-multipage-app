# booksbynik-multipage-app
# BooksByNIK PWA  **Architecture:** MPA.  **Backend:** G-Script Serverless API. Data uses 3 G-Sheets (Catalog, Orders, Profiles) via 3 APIs (`script.js`).  **Login:** Mandatory. Name/Number requires re-login to change. Language is editable via `profile.html`.  **Files:** Core HTML, `script.js`, `manifest.json`.
